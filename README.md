## 开发与运行

### 后端项目

* 仓库地址：
https://github.com/kylin-lawliet/blackcat-file-plex.git


### 安装依赖
```
npm install
```

### 开发模式调试
```
npm run dev
```

### 打包项目
```
npm run build
```

### 运行项目
```
docker build -t vue:v1 .
docker run --name vue -d -p 80:80 vue:v1
```
